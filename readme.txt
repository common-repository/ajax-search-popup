Search By Post And Page On KeyUp.

== Description ==
This plugin to search by Post And Page on KeyUp with Ajax.
 Live Ajax Search.
 Easy To Customize.
Quick and easily access to information.


= Usage =

1. To get started, click on the sub-menu Search Popup > Setting. 
2. You can use shortcode to run searchform.
2. Example: [ajax_popup_search]
3. Add to widget

== Installation ==

1. Upload the 'ajax-search-popup' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Add the shortcode to your page.
